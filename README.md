# PSite

Personal portfolio site built with Vue 3, TypeScript, Vite, Vue Router, and Tailwind CSS. Three sections: About, Education & Certificates, and Projects.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Starts the dev server at http://localhost:5173.

## Build

```bash
npm run build
```

Type-checks and builds the production bundle into `dist/`.

## Preview production build

```bash
npm run preview
```
